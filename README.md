# E-commerce-Shipment-Analysis
## Project Overview
E-commerce shipment data analysis using Python and Power BI to identify delivery delays and optimize logistics performance

## Objectives
- Analyze shipment delays
- Identify key delivery performance issues
- Visualize shipment efficiency and trends
- Support decision-making through dashboards

## Dataset
- E-commerce shipment data
- Includes order details, shipment mode etc.

## Tools & Technologies
- Python (Pandas, NumPy)
- Data Visualization: Matplotlib, Seaborn
- Power BI
- Power Query (Data Transformation)

## Key Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Shipment delay trend analysis
- Performance comparison across shipment modes

## Dashboard
Designed interactive Power BI dashboards to track:
- Delivery time trends
- Order volume
- Shipment delays

## Key Insights
- Identified major causes of delivery delays
- Highlighted regions with logistics inefficiencies
- Provided recommendations for improving shipment performance

- Python analysis files
